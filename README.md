## Descrição do Projeto

O **ConectaLocal** é uma plataforma web desenvolvida para conectar pequenos empreendedores locais a potenciais clientes da região. O sistema permite que empreendedores divulguem seus produtos ou serviços por meio de anúncios, facilitando a busca por oportunidades e serviços locais.

Este projeto está sendo desenvolvido como parte do Projeto Final da Disciplina, seguindo a proposta de construção de um MVP (Minimum Viable Product) funcional.

---

## Tema do Projeto

**Plataforma de divulgação de serviços e produtos para pequenos empreendedores locais.**

O objetivo é oferecer um ambiente simples onde empreendedores possam cadastrar anúncios de seus serviços ou produtos, enquanto clientes podem visualizar e buscar essas oportunidades.

---

## Delimitação do Escopo

Para esta primeira etapa do projeto, o escopo foi limitado às funcionalidades essenciais necessárias para validar a proposta do sistema.

### Funcionalidades previstas para o MVP

#### Usuários
- Cadastro de usuário
- Login de usuário
- Visualização de usuários cadastrados

#### Anúncios
- Cadastro de anúncios
- Listagem de anúncios
- Edição de anúncios
- Exclusão de anúncios
- Visualização dos anúncios cadastrados por um usuário

### Funcionalidades fora do escopo desta etapa

As seguintes funcionalidades não serão implementadas neste MVP:

- Recuperação de senha
- Upload de imagens
- Sistema de avaliações
- Sistema de mensagens entre usuários
- Geolocalização
- Painel administrativo
- Sistema de pagamentos
- Integrações com APIs externas
- Notificações em tempo real

A exclusão dessas funcionalidades tem como objetivo manter o projeto compatível com o prazo de desenvolvimento e com os requisitos acadêmicos da disciplina.

## Objetivo do MVP

O MVP deverá permitir que um empreendedor:

1. Crie uma conta no sistema;
2. Faça login;
3. Cadastre anúncios de serviços ou produtos;
4. Visualize seus anúncios;
5. Edite anúncios cadastrados;
6. Exclua anúncios;
7. Tenha seus dados armazenados em banco de dados PostgreSQL.

Os clientes poderão acessar a plataforma e visualizar os anúncios disponíveis.

---

## Justificativa da Escolha do Tema

Pequenos empreendedores frequentemente possuem dificuldades para divulgar seus serviços e produtos de maneira acessível. O ConectaLocal busca oferecer uma solução simples para aproximar empreendedores e clientes, promovendo a divulgação local e facilitando o acesso a serviços da comunidade.
